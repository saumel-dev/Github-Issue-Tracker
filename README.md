<h3>1. What is the difference between var, let, and const?</h3>
<br>
answer:
  The difference between var, let and const is in their scope and behavior.
var declears variables with global scope and allows redeclearation and updates within the same scope
let declears variables with block scope and allows updates but not redeclearation within the same block
const declears variables with block scope and doesn't allows redeclearation/reassign and doesn't allow updates

<h3>2. What is the spread operator (...)?</h3>
<br>
answer:
  The spread operator (...) in JavaScript is an ES6 syntax that provide a simple and expressive way to expand elements from arrays, strings or object. It expands elements of arrays and strings or properties of objects into individual values.

<h3>3. What is the difference between map(), filter(), and forEach()?</h3>
<br>
answer:
  Javascript provides several ways to manipulate array, and the mostly commons are map, filter, and forEach. The forEach method executes a provided function once for each array element. The map method creates a new array populated with the results of calling a provided function on every element in the calling array. It is used to transform data. The filter method creates a new array with all elements that pass the test implemented by the provided function. It is used to include only certain elements from the original array.
